# kubernetes_lokal

Lern- und Umsetzungsprojekt fuer Kubernetes, vCluster, Helm und Terraform mit dem Ziel, eine dauerhafte Testumgebung fuer Anwendung A lokal vorzubereiten und spaeter auf AKS zu betreiben.

## Enthalten

- Architektur- und Planungsdokumente
- lokales Setup mit `kind` und `vCluster`
- minimales Helm-Chart fuer `app-a`
- Terraform-Struktur fuer Plattform, Shared Services und State-Ressourcen

## Naechste Schritte

1. Lokalen Host-Cluster mit `kind` erstellen.
2. `vCluster` fuer Anwendung A anlegen.
3. Helm-Chart lokal deployen.
4. Terraform-Stacks fuer Azure weiter ausarbeiten.
5. Repository nach GitHub pushen.
